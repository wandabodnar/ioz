
# From spatial data to published interactive maps using R

**Training date:** 21/01/2026  
**Led by:** Wanda Bodnar  

This repository supports the training **“From spatial data to published interactive maps using R”**, covering:

1) Creating & loading spatial data  
2) Publication‑ready static maps  
3) Interactive maps with mixed data sources  
4) Publishing maps online

## Pre‑session checklist

### 1) Install R and RStudio
- Install **R first**, then install **RStudio Desktop**.

### 2) Install required R packages
- Open the provided workshop script (or use the block below) and run it in RStudio.

### 3) GitHub setup
- Create a GitHub account
- Install **GitHub Desktop**
- (Optional) If using Copilot in RStudio: Tools → Global Options → Copilot → Enable
